# Over 30 new plugins join the Cursor Marketplace · Cursor

Last month, we launched the [Cursor Marketplace](https://cursor.com/blog/marketplace) to make it easier to connect Cursor to your existing tools and knowledge.

Today, we're adding more than 30 new plugins from partners such as Atlassian, Datadog, GitLab, Glean, Hugging Face, monday.com, and PlanetScale. Cursor can now read from, write to, and take actions across more of your stack.

Over 30 new plugins join the Cursor Marketplace

You can install plugins on the [Cursor Marketplace](https://cursor.com/marketplace), or build your own and [share them](https://cursor.com/marketplace/publish) with the community. On Teams and Enterprise plans, Admins can also create [team marketplaces](https://cursor.com/docs/plugins#team-marketplaces) for distributing and managing private plugins.

We've found that what matters most for an agent's success is access to the right tools and context. Plugins provide that by bundling capabilities like MCPs with skills that instruct the agent on how to use it. Since the initial release of Cursor Marketplace, users have reported that this combination is much more powerful than MCPs on their own.

The plugins we're introducing today span the entire development workflow, equipping agents to work more independently. Two broad buckets we're particularly excited about are infrastructure and productivity tools.

### Infrastructure

Use natural language to query logs, metrics, traces, and dashboards with the [Datadog plugin](https://cursor.com/marketplace/datadog).

View and manage issues, merge requests, pipelines, and repositories with the [GitLab plugin](https://cursor.com/marketplace/gitlab).

Discover and apply schema recommendations, improve query performance, and work with your data in natural language with the [PlanetScale plugin](https://cursor.com/marketplace/planetscale).

### Productivity

Search and manage issues, generate status reports, and convert specs into backlogs with the [Atlassian plugin](https://cursor.com/marketplace/atlassian).

Search company knowledge, and find project owners and stakeholders with the [Glean plugin](https://cursor.com/marketplace/glean).

Access and manage project boards and tasks with the [monday.com plugin](https://cursor.com/marketplace/monday).

## Tools for automations

Most plugins contain MCPs that cloud agents can use when kicked off manually or invoked automatically. You can use [Cursor Automations](https://cursor.com/blog/automations) to build always-on agents that run on schedules or are triggered by events. They follow your custom instructions, and can use tools like the Datadog MCP to investigate logs depending on the task they're configured to complete.

Try out the new plugins at [cursor.com/marketplace](https://cursor.com/marketplace), or learn more in the [docs](https://cursor.com/docs/plugins).
