## LangSmith Updates

### 🔔 Fleet On-Call Copilot

A prebuilt agent template that works through your code, traces, and runbooks to triage alerts and draft updates for your review.

[Try On-Call Copilot →](https://smith.langchain.com/agents/templates?viewingTemplateId=on-call-copilot)

### 💻 Computer Use in Fleet

Agents can now use an isolated virtual computer for code, files, and authenticated API calls.

[Learn more →](https://docs.langchain.com/langsmith/fleet/computer-use/)

### ▶️ New UI for Voice Traces

Debug audio faster with an inline audio player and see exactly which span was active at each moment.[Learn more →](https://docs.langchain.com/langsmith/trace-voice-fundamentals/)

### 📊 Experiment Status Tracking

See live progress for experiments started from the LangSmith UI or SDK.

[Learn more →](https://docs.langchain.com/langsmith/analyze-an-experiment)

### 💬 Native Slack Integration for LangSmith Engine

Send Slack notifications for Engine Issues.

[Learn more →](https://docs.langchain.com/langsmith/engine#notify-a-slack-channel)

## Open Source

### Deep Agents Rubrics

`RubricMiddleware` helps Deep Agents evaluate their own work and keep iterating until the task meets your criteria.

[Read the blog →](https://www.langchain.com/blog/introducing-rubrics-for-deepagents)

### Programmatic Subagents

Dispatch subagents from interpreter code to fan out work and synthesize results.

[Read the docs →](https://docs.langchain.com/oss/python/deepagents/programmatic-subagents/)
