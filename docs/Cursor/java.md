# Improving Java support in Cursor · Cursor

We're making a major investment to improve Java support in Cursor.

This includes building a dedicated team to speed up the Java Language Server Protocol (LSP) and contributing open-source improvements back to the VS Code ecosystem.

We're partnering with LSP authors and contributors from Red Hat, Microsoft, IBM, and more. Early improvements have shown 10% faster project imports, and reducing debugger scans from 5+ seconds to under 200 ms in large codebases.

> Cursor's focus on LSP performance bottlenecks and project initialization directly addresses the scaling challenges that enterprise Java developers face daily. These improvements will benefit the entire VS Code Java ecosystem.
