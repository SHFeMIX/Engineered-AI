## Breakout session

### Untrusted code, unprecedented speed: High-velocity runtimes for AI agents

LangChain will be featured in a breakout session at Google Cloud Next on building high-performance, secure runtimes for AI agents.

The session explores how to scale agent infrastructure while safely running untrusted, LLM-generated code, covering techniques like subsecond cold starts for massive concurrency and using gVisor for kernel-level isolation in zero-trust environments. It will also highlight how these approaches power real-world AI agent systems in production.

Victor Moreira joins as a customer speaker, sharing how LangChain leverages this architecture to run and scale autonomous agents securely on Google Cloud.

Speakers:

- Victor Moreira, Forward Deployed Engineer @ LangChain
- Jeremy Olmsted-Thompson, Principal Engineer @ Google Cloud
- Brandon Royal, Product Manager @ Google Cloud

## LangSmith on Google Cloud Marketplace

LangSmith is now available in the Google Cloud Marketplace, making it easier for teams building on Google Cloud to procure and deploy LangChain’s platform for AI agents.

With Marketplace availability, organizations can purchase LangSmith through their existing Google Cloud account, apply it toward committed cloud spend, and simplify procurement through consolidated billing.

LangSmith helps teams debug, evaluate, and monitor AI applications, providing visibility into agent performance and production behavior.
