# The Agent Client Protocol Overview

With the explosion of AI coding agents—Claude Code, Gemini CLI, OpenCode, Goose, Codex CLI, we've entered an era where every major AI lab ships their own terminal-based coding assistant. But there is one problem. Each agent has its own events or outputs. The \[Agent Client Protocol (ACP)\](https://agentclientprotocol.com/get-started/introduction) is an an open standard abstracts the events and outputs of AI agents and provides a common interface for editors to interact with them. Similar to MCP but for agent to client (UI) communication. ACP is a JSON-RPC 2.0 protocol with a simple premise: 1. \*\*Clients\*\* (editors like Zed, JetBrains or Obsidian) manage the user environment 2. \*\*Agents\*\* (Claude Code, Gemini CLI, OpenClaw (prev. Clawdbot)) Thinking and tool execution 3.
